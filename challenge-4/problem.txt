# Challenge #4: Design using CRC cards 

## Instructions

Design a solution for the Tetris game using CRC cards.

## Scoring

You get points for: solution that fits the problem, simplicity of the solution, and following SOLID principles. 

You will get:

* 0 points if no pull request 
* 1 point if you make a pull request even if there's no solution
* 2-40 points if you make a pull request with a partial or complete solution
